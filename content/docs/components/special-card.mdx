##Usage

import SpecialCard from '@/components/magicui/SpecialCard';

const cardData = {
  coverImage: 'cover.jpg',
  titleImage: 'title.png',
  characterImage: 'character.png',
};

export default function SpecialCardDemo() {
  return <SpecialCard card={cardData} />;
}

## Props

##SpecialCard

| Prop          | Type      | Description                                        | Default                                  |
| ------------- | ------    | -------------------------------------------------- | ---------------------------------------- |
| card          | CardProps | The data for the card including images             | -                                        |

## CardProps

| Prop             | Type   | Description                           | Default |
| ---------------- | ------ | ------------------------------------- | ------- |
| coverImage       | string | URL or path to the cover image.       | -       |
| titleImage       | string | URL or path to the title image.       | -       |
| characterImage   | string | URL or path to the character image.   | -       |
